## v1.0.0 - Initial Release - [2026-02-04]

### `Overview`

Caymanflow is a Nextflow pipeline for annotating shotgun metagenomic paired-end reads using Cayman. This pipeline provides quality control with fastp (optional) and direct annotation of metagenomic reads.
